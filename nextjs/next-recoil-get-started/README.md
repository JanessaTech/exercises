## How to create this project from 0

```
    npx create-next-app@latest (choose default options)
    enter the dir just created
    npx shadcn-ui@latest init  (choose default options)
    npm install recoil
```

## How to install dependences

```
npm install
```

## How to run

```
  npm run dev
  npm run build
  npm run start
```
